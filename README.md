# A simple timer/counter for linux terminal.

### prerequisites:
- [sfml](https://www.sfml-dev.org/tutorials/2.6/start-linux.php#installing-sfml)

### Build
```sh
git clone https://github.com/assaidy/timer
cd timer
./build.sh
```

### Usage:
```sh
$ timer
$ timer [DURATION]
         DURATION: '$h $m $s'
         DURATION:  $h
         DURATION:  $m
         DURATION:  $s
```
